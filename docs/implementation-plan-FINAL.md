## üé¨ ANIMATION REQUIREMENTS (ALL VARIATIONS)

### Parallax Effects (GSAP ScrollTrigger)

**Hero Section:**
```javascript
// 3 parallax layers minimum
backgroundLayer: { speed: 0.3 },
imageLayer: { speed: 0.6 },
textLayer: { speed: 1.0 }
```

**Section Transitions:**
```javascript
// Sections fade in + slide up
gsap.to(section, {
  y: 0,
  opacity: 1,
  duration: 0.8,
  ease: "power2.out",
  scrollTrigger: {
    trigger: section,
    start: "top 80%"
  }
});
```

### Framer Motion Animations

**Stagger Reveals:**
```typescript
const container = {
  hidden: { opacity: 0 },
  visible: {
    opacity: 1,
    transition: {
      staggerChildren: 0.1
    }
  }
};

const item = {
  hidden: { y: 40, opacity: 0 },
  visible: {
    y: 0,
    opacity: 1,
    transition: { duration: 0.6 }
  }
};
```

**Hover Interactions:**
```typescript
<motion.div
  whileHover={{ scale: 1.03, y: -8 }}
  transition={{ duration: 0.3 }}
>
  <PricingCard />
```

```typescript
const lenis = new Lenis({
```
