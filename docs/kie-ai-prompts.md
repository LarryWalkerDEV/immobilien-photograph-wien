# kie.ai Image Generation Prompts for SEO Pages

This document contains optimized prompts for generating hero images and portfolio images for each SEO page using the kie.ai API.

## General Settings for All Prompts

- **Model:** kie.ai 4.5 (text-to-image)
- **Aspect Ratio:** 16:9 for hero images, 4:3 for portfolio images
- **Quality:** 8K, photorealistic
- **Style:** Professional real estate photography, bright, inviting

## Usage Instructions

1. **Copy the prompt** for the specific page you're generating images for
2. **Access kie.ai API** using the configured API key
3. **Set parameters:**
   - Model: 4.5 for images, Kling 2.6 for video
   - Aspect ratio: 16:9 for heroes, 4:3 for portfolio
   - Quality: Maximum available
4. **Generate and download** the images
5. **Save to:** `/public/generated/` with naming convention: `{page-slug}-hero.jpg`
6. **Update manifest.json** with new image URLs

## Naming Convention

- Hero images: `{page-slug}-hero.jpg`
- Portfolio images: `{page-slug}-portfolio-{n}.jpg`
- Videos: `{page-slug}-video.mp4`

Example:
- `360-grad-rundgaenge-immobilien-wien-hero.jpg`
- `360-grad-rundgaenge-immobilien-wien-portfolio-1.jpg`
- `360-grad-rundgaenge-immobilien-wien-video.mp4`
